# Pleme(콕 찌르기)
알림 톡 기능을 활용한 메시지 소통 앱 구현
  
## 프로젝트 소개
'뉴트로'라는 개념이 주목받기 시작하면서 페이스북의 사라진 '콕 찌르기' 기능을 활용하여 개발한 새로운 SNS 어플리케이션입니다.

## 프로젝트 정보

### ⚙️ 개발 환경
- `Java, Kotlin`
- **Framework** : AndroStudio
- **DataStore** : Firestore
- **DataBase** : Firebase
- **Design** : Figma
